# EntroPy

EntroPy is a Python 3 package for computing several entropy metrics of time series.

It currently includes
- `perm_entropy`: Permutation entropy (Bandt and Pompe, 2002)


# Installation

## Dependencies
- NumPy
- SciPy

`python setup.py develop`
